El módulo se almacena en una carpeta con el nombre corto del módulo

El módulo esta compuesto por dos archivos que contiene el nombre corto del módulo

Ejemplo para un módulo llamado "ejemplo_corto"

carpeta: ejemplo_corto
archivo1: ejemplo_corto_admon.php
archivo2: ejemplo_corto_vista.php

archivo1 (ejemplo_corto_admon.php)
Descripción: Este archivo contendrá todas las instrucciones necesarias para adminsitrar la información especializada que pretenda manejar.

archivo2 (ejemplo_corto_vista.php)
Descripción: Este archivo contendrá todas las instrucciones necesarias para visualizar la información especializada en la vista final del portal.

Para una explicación y ayuda más extensa de como realizar módulos nuevos para Nazep, visitar el portal web oficial.
